收获，成长

2016年悄然已去，回想一下下，依然有不少收获。

技术

今年一来就是开始重构项目，主要是在项目中使用AndroidAnnotation这个注解框架，在项目重构的过程中慢慢的了解了其他类似的开源的框架，也开始阅读开源框架，然后把自己的对框架的理解写在《PSS技术小屋里》。在工作之余也学习一些当下流行的设计模式以及框架等，并学习使用那些框架搭建自己的个人项目。

今年接触到不少程序开发语言，框架，开发模式等，开发框架比如SpringMVC，PlayFramework，YII，ThinkPHP等， 了解移动混合开发，Web后台开发，App直播技术，微信小程序开发等。

目前主要是靠着兴趣去学习，但是我知道学习一门技术还是要对这门技术有全局的了解，它能解决什么问题， 所以还需改变认识，深入研究。

沟通

今年经历了多次Share, 以及带了两次Android training之后，个人感觉在做演讲的时候没有以前那么的紧张了，自己的想表达的东西，也能很好的表达给大家。

虽然现在还是不是直接和客户直接沟通，但是如果有问题，还是会写邮件给客户，经过一年的锻炼，个人感觉，写邮件的速度和质量还是在不断提高。

培训技能

今年是我第一次带实习生，说实话，刚开始有还是有些紧张，原因是自己已经很长时间没有接触 Java web怕到时候“带坏了”小朋友。正是因为有这方面的顾虑，所以我在私下会学员们一起学习要们要学习的内容。如果遇到学员提到过的问题，我都会记下来，然后回去研究研究。整个培训下来我自己也学到不少东西。

经过这个培训，让我熟悉了整个培训流程，在技术方面接触或者回顾了平时很少接触到的知识点，同时也认识了一些新同事。

累并快乐

今年需要经常的回老家，因为小孩不在身边，来回有时候确实很累，但是每次回家很够看到小孩那开心的笑容，很够看到他在一天天的长大，一天天的进步，其他的也就都觉得无所谓了，这也许就是做父母的感受吧。

困难, 挑战, 不足

困难和挑战并存的一年。

其实最大的困难还是来自技术上的，PSS项目正处在迭代开发过程中，客户那边也有开发，他们给人感觉很厉害，很有经验，很老道，我们做事需要处处小心。 不过在和他们合作的过程中学到了不少知识。

一直就是觉得自己学的东西比较杂乱，而且研究都不够升入。

另外一个就是个人的瓶颈了。平时表现不算活跃，私下也是一个安静的人，不善于与人交心的去沟通。 虽然在技术方面已经积累的一些经验，但是在人员和团队方面的管理和凝聚方面，着实没什么建树。

总结

一年又过去了，当想起写总结的时候，脑子里一片空白，感觉没有什么内容可说的，唯有一声感叹，一年一年过的好快！回想了很久，总结一下这一年，2016其实很平静, 平静地自始至终都没有什么轰轰烈烈的情节， 没有大起大落，学习，生活，还是按往常的进行，遇到问题，解决问题，遇到困难，想办法克服困难。 2016年于我来说还算是充实的，项目在有条不紊的进行，认识和很多新同事，学习了一些新的技术，同时也很高兴有很多人来听我的分享。 在这一年里有成长有提高，也发现了更多的不足，还寄希望于未知的2017年去改进。

展望2017

2017年我其实也没有多大的设想，唯一的就是希望小孩能够健健康康的成长，家人能够健健康康的。至于工作，希望PSS项目重构顺利，项目能够有条不紊的发展下去，我也会不断学习新的技术，不断与大家分享。
